# TP 2 - Métodos numéricos

## Como compilar

El programa se compila ejecutando el comando 

	make

## Como ejecutar

El comando para correrlo es: 

	$ ./recortador archivoEntrada archivoComparativo archivoResaizeado k

Donde 

- archivoEntrada es la imagen original
- archivoComparativo es la imagen mas cercana a la original contra la que se puede comparar la resaizeada
- archivoResaizeado es la imagen reducida para que se pueda usar con el k dado y comparar
- metodo es un entero que indica cuanto zoom se harà en la foto
